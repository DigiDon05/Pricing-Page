# 💳 Modern Animated Pricing UI

A sleek and modern **Pricing Card UI** built using **HTML & CSS**, featuring smooth animations, glassmorphism design, and responsive layout.

---
---

## 🚀 Features

✨ Glassmorphism UI (blur effect)
✨ Animated gradient border
✨ Smooth hover animations
✨ Highlighted active plan
✨ Clean and minimal design

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox, Animations, Glassmorphism)

---

## 📂 Project Structure

```bash
📁 pricing-ui
 ┣ 📄 index.html
 ┣ 📄 style.css
 ┗ 📄 preview.png / preview.gif
```

---

## 🎯 How It Works

* Flexbox is used to align pricing cards
* Glass effect is created using `backdrop-filter`
* Gradient border uses CSS animations
* Hover effects use `transform` and `box-shadow`
